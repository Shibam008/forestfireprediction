FWI Prediction System

This web application predicts the Fire Weather Index (FWI) using meteorological and environmental data from the Algerian Forest Fires Dataset. The Fire Weather Index is an important indicator used to estimate the potential intensity of forest fires based on weather conditions. By providing the environmental parameters below, the machine learning model estimates the FWI value which indicates the fire risk level.
Input Parameters Explanation:
  • Temperature – Air temperature in degrees Celsius.
  • RH – Relative Humidity (%), representing moisture in the air.
  • Ws – Wind speed (km/h), affecting fire spread.
  • Rain – Amount of rainfall (mm).
  • FFMC – Fine Fuel Moisture Code indicating surface fuel dryness.
  • DMC – Duff Moisture Code representing moisture in decomposed organic material.
  • ISI – Initial Spread Index estimating the expected rate of fire spread.
  • Classes – Fire occurrence classification (fire or not fire).
  • Region – Geographic region indicator from the dataset.
